
## Remediation

### Quick Fix — Restore Service

Take these steps to restore service immediately while investigating root cause.

**Step 1: Attempt a rolling restart (safe, no downtime)**

```bash

# Monitor the rollout
kubectl rollout status deployment/[service-name] -n [namespace] --timeout=5m
```

**Step 2: If restart doesn't help — rollback last deployment**

